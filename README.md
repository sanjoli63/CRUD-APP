## Features
```
1. List of All Transactions
2. Add new Transactions
3. Recent week Expense Statistics
4. Delete your Transactions
```
### Widgets Implemented
```
1. Fractionally Sized Box
2. Bottom Model Sheet
3. Expanded
4. List View Builder
5. Mapping List to build Widgets
```
